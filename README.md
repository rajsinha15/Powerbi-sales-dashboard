# Power BI Sales Dashboard

## 📊 Project Overview
This project is an interactive Power BI dashboard built to analyze e-commerce sales data, including revenue, profit, quantity, and customer insights.

## 📈 Key Features
- Sales analysis by state
- Monthly profit trends
- Category-wise quantity distribution
- Customer-wise sales insights
- Payment mode analysis
- Sub-category profit breakdown

## 🛠️ Tools Used
- Power BI
- Data Analysis
- Data Visualization
